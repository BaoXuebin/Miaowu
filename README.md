## 安装方法

1. 打开 `更多工具` >> `扩展程序`, 或者直接 chrome 地址栏输入 `chrome://extensions/` 回车。
2. 打开 `右上角` 的 `开发者模式`。
3. 点击 `加载已解压的扩展程序` 打开对应文件夹。

## 使用方法

**目前只能来当字典用**

点击图标，或者快捷键 `Ctrl+Shift+O` 打开。如果网页中有选中内容，则默认直接翻译对应的选中文本。
